From Tools to Impact

Technology doesn’t change classrooms—people do. This site turns AI into a thinking partner, not a crutch.

What is this place? How do I use it?

This is a living, do-it-now toolkit. Every section has a clear purpose, plain steps, and an export. You’ll frame a smart prompt (RISEN), plan a habit, prep data correctly, choose the right platform on purpose, tell the story, and sprint for 10 days.

  1. Start Guided Mode (pick your role). It fills good defaults and jumps you section→section.
  2. Planner turns a tool into a weekly habit with owner, cadence, and outcome.
  3. Data Prep Coach translates “squishy idea” → AI-ready spec + CSV + checklist.
  4. AI Platform Finder selects categories + gives search strings and a rubric (examples included).
  5. Story makes a board-ready 3-liner; 10-Day Plan seeds execution.

Badges you’ll see: [Drop-in AI] output is designed to paste directly into a model; [Human workflow] is for teams/systems.

Data Prep Coach [Drop-in AI]

Turn a messy idea into a clean, AI-ready spec + CSV + checklist.

What’s a field?A column in your data table (e.g., student_id, date, status). What’s a join key?A unique ID that lets you connect tables (e.g., student_id, campus_id). What’s grain?The level of each row (e.g., student/day, campus/week).

Example (attendance): fields → student_id,date,present,period,reason_code | grain → student/day | join keys → student_id
